# Zerodha Clone

A full-stack stock trading platform inspired by Zerodha.

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- Axios
- Bootstrap

## Features

- User Authentication
- Buy Stocks
- Sell Stocks
- Holdings Management
- Orders Tracking
- Funds Management
- Portfolio Dashboard
- Real-time Data Visualization

## Installation

```bash
git clone https://github.com/AyushKumar-12345/zerodha-clone.git
cd zerodha-clone
npm install
```
